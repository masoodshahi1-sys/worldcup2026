## مرحله ۱ — ساخت Firebase (رایگان، ۵ دقیقه)

1. برو به **https://console.firebase.google.com**
2. کلیک کن روی **"Create a project"**
3. نام پروژه: `worldcup2026` (یا هر چیزی)
4. Google Analytics رو **غیرفعال** کن → **Create project**
5. بعد از ساخت، کلیک کن روی **آیکون `</>`** (Web App)
6. نام اپ: `wc2026` → **Register app**
7. کد `firebaseConfig` رو **کپی** کن

### فعال کردن Firestore:
- از منوی سمت چپ: **Firestore Database**
- کلیک: **Create database**
- انتخاب: **Start in test mode**
- Region: **europe-west3** (Frankfurt) ← نزدیک‌ترین به ایران
- کلیک: **Enable**

### تنظیم Rules (مهم!):
در Firestore → Rules، این رو جایگزین کن:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
```
کلیک **Publish**

## مرحله ۳ — آپلود روی GitHub

1. برو به **https://github.com** → ثبت‌نام کن (اگه نداری)
2. کلیک: **New repository**
   - نام: `worldcup2026`
   - Public یا Private: هر کدام
   - **Create repository**
3. پوشه `wc2026` رو روی کامپیوترت بذار
4. Terminal باز کن داخل پوشه:

```bash
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/USERNAME/worldcup2026.git
git push -u origin main
```

> اگه Git نداری: **https://desktop.github.com** رو نصب کن — گرافیکیه، راحت‌تره.

---

## مرحله ۴ — Deploy روی Vercel (رایگان)

1. برو به **https://vercel.com** → با GitHub لاگین کن
2. کلیک: **"Add New Project"**
3. Repository `worldcup2026` رو انتخاب کن → **Import**
4. Framework: **Create React App** (اتوماتیک تشخیص می‌ده)
5. کلیک: **Deploy**
6. بعد از ۲-۳ دقیقه → لینک آماده‌ست! مثلاً:

```
https://worldcup2026.vercel.app
```
